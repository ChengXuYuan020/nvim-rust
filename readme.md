操作备忘：
1. \<leader\> -- 空格
2. C -- Ctrl, A -- Alt, S -- Shift
3. \<CR\> -- 回车
4. 自动保存的触发：离开 i 模式，buffer 栏上的绿色小圆点表示文件未写入
